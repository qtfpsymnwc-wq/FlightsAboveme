About FlightsAboveMe
What this site is, where the data comes from, and what to expect

FlightsAboveMe is a lightweight “what’s overhead?” flight display. It’s built for quick checks on your phone and for always‑on wall boards and TVs using Kiosk Mode.

What you’ll see

  • Nearby aircraft with altitude, speed, distance, and direction
  • Basic aircraft details when available (type/model/operator)
  • Simple labels like climbing/descending/cruising based on vertical speed

Data sources

FlightsAboveMe blends multiple aviation data sources to improve reliability and add aircraft details:

  • OpenSky Network — primary live state vectors when available
  • adsb.lol — fallback live aircraft state data
  • AeroDataBox — aircraft enrichment (type/model/operator details)

Limitations

Coverage and accuracy vary by region and aircraft equipment. Some flights may be delayed, incomplete, or missing. If location access is disabled, you can still view flights by entering a U.S. ZIP code.
