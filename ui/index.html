What is FlightsAboveMe?

FlightsAboveMe shows aircraft currently flying near your location using live flight tracking data. It’s designed for quick “what’s overhead?” checks and for always-on displays.

  • See the closest flights near you, including altitude, speed, distance, and direction.
  • Tap Kiosk Mode for an ad-free, full-screen flight board for TVs and tablets.
  • If you don’t allow location access, you can enter a U.S. ZIP code to view flights near that area.
