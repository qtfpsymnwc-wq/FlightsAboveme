Squawk codes explained
What transponder codes mean and why they matter

A squawk is a 4‑digit transponder code (Mode A) used by radar and air traffic control systems. Controllers assign squawks so aircraft can be identified and tracked on ATC displays.

Common emergency squawks

  • 7700 — general emergency
  • 7600 — radio communications failure
  • 7500 — unlawful interference (hijacking)

How squawks work

Squawk codes are entered on the aircraft's transponder and transmitted to ground radar/ATC systems. Most squawks are discrete (uniquely assigned to one aircraft at a time) so controllers can correlate a radar target with a specific flight.

Why the digits look “odd”

Squawk codes are written as four digits, but each digit is octal (0–7). That's why you won't see 8 or 9 in a squawk code.

Mode A vs Mode C vs Mode S

  • Mode A: the 4‑digit squawk code
  • Mode C: adds pressure altitude (often what tracking sites display as altitude)
  • Mode S / ADS‑B: adds a unique aircraft address (ICAO24) and, with ADS‑B, position and other data

Conspicuity codes (varies by country)

In some situations aircraft may use a standard “conspicuity” code when not assigned a discrete squawk. These common codes vary by country and airspace rules, so FlightsAboveMe focuses on explaining the squawk when it is available rather than guessing what it should be.

Why squawk may be missing

Not all public feeds provide squawk consistently, and some sources omit it for privacy or data-quality reasons. When it’s available, FlightsAboveMe may display it and/or a plain‑English label.

Why squawk digits are 0–7 only

Squawks are octal — each digit ranges from 0 to 7. That’s a historical design from early transponder systems and radar encoding. So you will never see an 8 or 9 in a standard squawk.

Discrete vs “conspicuity” codes

A discrete code is assigned to a specific aircraft by ATC. A conspicuity code is a common code used to be visible to radar when not receiving a discrete assignment. Examples include 1200 for VFR in the United States and 7000 in many European regions (rules vary by country).

How squawks relate to Mode A/C/S

Mode A carries the squawk code. Mode C adds altitude reporting. Mode S adds a unique aircraft address (ICAO24) and supports richer messaging. ADS‑B builds on Mode S to broadcast position and velocity.

Why squawk might be missing or change

Some networks suppress squawks for privacy or safety reasons, and not all receivers decode every message reliably. Squawks can also change during a flight when handed off between controllers or when entering controlled airspace.
