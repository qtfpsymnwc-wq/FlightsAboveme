Build Your Own ADS‑B Receiver
A practical, beginner-friendly setup using a Raspberry Pi + RTL‑SDR

Many aircraft broadcast their identity, altitude, speed, and position using ADS‑B on 1090 MHz. With a small computer and an inexpensive radio receiver, you can collect these broadcasts yourself and build a local “what’s overhead?” feed.

What you’ll need

  • Raspberry Pi (3B+ or newer recommended)
  • MicroSD card (16GB+)
  • RTL‑SDR USB dongle (with a suitable oscillator/TCXO if possible)
  • 1090 MHz ADS‑B antenna (or a tuned antenna designed for 1090 MHz)
  • Coax + adapters (keep cable runs short when possible)
  • Power supply and (optional) a case

Quick setup overview (Raspberry Pi)

  1. Install Raspberry Pi OS on a microSD card and boot the Pi.
  2. Plug in the RTL‑SDR dongle and connect your 1090 MHz antenna.
  3. Install a decoder (common choices: readsb or dump1090).
  4. Open the local web map (often available on your network at the Pi’s IP address).
  5. Optional: feed a public network so your receiver helps improve coverage for everyone.

1090 MHz antenna placement tips

  • Higher is better. ADS‑B is line‑of‑sight — elevation increases range dramatically.
  • Use good coax. Cheap cable can lose a surprising amount of signal at 1090 MHz.
  • Weatherproof outdoor runs. If mounting outside, protect connections from water.
  • Consider filtering. In dense RF areas, a 1090 MHz filter or an LNA near the antenna can help.

What range should you expect?

Range depends on antenna height and local terrain. With a modest antenna indoors you may see 50–100 miles at cruise altitudes. With an outdoor antenna mounted high, 150–250+ miles for high‑altitude aircraft is common. Low‑altitude aircraft will always have shorter range.

Feeding data (optional)

Many communities and research networks accept receiver feeds. Contributing can improve coverage in rural areas and help scientific or safety projects. Always review each network’s terms before sharing data.

Placement tips (this matters a lot)

  • Higher is better: a window, attic, or mast usually improves range.
  • Line of sight: buildings and hills reduce reception.
  • Avoid interference: keep the dongle and antenna away from noisy power supplies and routers.

Software options

Most DIY setups decode ADS‑B using a decoder like dump1090 (or modern variants like readsb), then optionally run a local web map and/or feed data to a network.

  • PiAware (FlightAware) — guided install, good dashboards
  • readsb + tar1090 — lightweight local decoder + web interface
  • Docker images — convenient if you’re already using containers

Feeding tracking networks (optional)

If you want to contribute to the community, many projects let you share your receiver data. Each has its own terms and setup process.

  • OpenSky Network
  • ADSBexchange
  • FlightAware
  • Flightradar24
